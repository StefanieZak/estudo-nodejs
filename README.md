# estudo-nodejs

## Project setup

### /estudo-nodejs:
```
yarn install or npm install
```

### /estudo-nodejs/api:
```
yarn install or npm install
```


### Compiles and hot-reloads for development
### /estudo-nodejs:
```
yarn dev or npm run dev
```
### /estudo-nodejs/api:
```
yarn start or npm run start
```

### Compiles and minifies for production
```
yarn build or npm run build
```
